# Notebooks

Jupyter notebooks for exploration, analysis, and development. Named with `{YYYY-MM-DD} - {description}` convention.

## Topics Covered

- Data processing and augmentation
- Model architecture prototyping (hierarchical classification, XGBoost baseline)
- DistilBERT pre-training (masked language model)
- Error analysis and model interpretability
- SSOC embedding visualization
- MCF data extraction and API testing
- Lambda deployment testing

## Plots

`Plots/` contains accuracy analysis charts (Top-10 accuracy by SSOC level, sample size effects).
